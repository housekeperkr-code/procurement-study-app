# 공공조달관리사 학습앱

아이패드에서 접속해 공부할 수 있도록 만든 정적 HTML 학습앱입니다.

## GitHub Pages 사용

이 저장소를 GitHub에 올린 뒤 `Settings > Pages`에서 `Deploy from a branch`를 선택하고, `main` 브랜치의 `/root`를 지정하면 됩니다.

접속 파일은 `index.html`입니다.

## 포함 파일

- `index.html`: GitHub Pages용 메인 파일
- `study_app_latest.html`: 로컬 확인용 원본 파일
- `assets/contract-rpg-pixel.png`: 절차 시뮬레이터 이미지
